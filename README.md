# Curso Python - Refatorando

Este repositório contém os códigos, exercícios e exemplos desenvolvidos durante o curso de [**Python para iniciantes**](https://www.youtube.com/watch?v=Jk4kuuwLThA&list=PLj7gJIFoP7jdirAFg-fHe9HKOnGLGXSHZ&index=1) do canal Refatorando no youtube.

O objetivo deste repositório é registrar a evolução no aprendizado da linguagem Python, praticando conceitos fundamentais de programação e organização de código.

---

## 📚 Conteúdo estudado

Durante o curso foram abordados conceitos como:

- Configuração do ambiente Python
- Variáveis e tipos de dados
- Operações matemáticas
- Strings
- Entrada e saída de dados (`input` e `print`)
- Estruturas condicionais (`if`, `else`)
- Estruturas de repetição (`for`, `while`)
- Estruturas de dados (`listas`, `tuplas`, `dicionários` e `sets`)
- Funções
- Tratamento de exceções
- Manipulação de arquivos
- Módulos e pacotes
- Introdução à Programação Orientada a Objetos (POO)
